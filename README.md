## Coverage

Generate coverage html [link](https://docs.djangoproject.com/en/4.0/topics/testing/advanced/#integration-with-coverage-py)

### Install coverage

```bash
$ pip install coverage
```


### Usage

```bash
$ coverage run --source="." manage.py test
$ coverage html
```
